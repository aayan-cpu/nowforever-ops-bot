About This Application

NowAndForever Ops Bot is an internal Google Chat application built exclusively for Now and Forever / Khawar and Sons gas station operations. It connects to 22+ Google Chat rooms and provides real-time monitoring, task tracking, and alert detection for field teams across Texas.

The bot uses the Google Chat API to receive and respond to messages posted in operational chat rooms. It classifies incoming messages, identifies urgent issues such as gas outages, equipment failures, or power cuts, and automatically creates and tracks operational tasks.

This application is not a consumer product. It is an internal operations tool used solely by authorized employees and managers of Now and Forever / Khawar and Sons.

Real-Time Monitoring

Reads and classifies messages from 22+ Google Chat rooms across all Texas sites, detecting urgent issues as they happen.

Task Tracking

Automatically creates and tracks operational tasks from chat messages, keeping managers aware of open issues across all locations.

Alert Detection

Identifies high-priority alerts such as no gas, power outages, and equipment failures and surfaces them for immediate action.

Google Chat Integration

Responds directly inside Google Chat rooms via the Google Chat API HTTP endpoint, accessible by tagging @NowForever Ops Bot.
