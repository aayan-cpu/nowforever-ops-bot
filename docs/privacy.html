1. Overview

NowAndForever Ops Bot is an internal Google Chat application operated by Now and Forever / Khawar and Sons. This Privacy Policy explains how the Application collects, uses, and protects information.

2. Who Uses This Application

This Application is not a consumer-facing product. It is used exclusively by authorized employees and managers of Now and Forever / Khawar and Sons. Access is restricted to internal personnel only.

3. Information We Access

The Application accesses the following through the Google Chat API:

  • Messages posted in designated operational Google Chat spaces
  • Space and membership metadata required to route and classify messages
  • User display names and Google Workspace IDs within the organization

The Application does not access Gmail, Google Drive, Google Calendar, or other Google Workspace services beyond Google Chat.

4. How We Use This Information

Information is used solely for internal operational purposes: classifying messages, creating and tracking tasks, detecting high-priority alerts, and generating internal dashboards. We do not use data for advertising or any purpose outside of operations management.

5. Data Storage

Processed data is stored in a database accessible only to authorized administrators within the organization. Data is not shared with any third parties.

6. Data Retention

Operational data is retained as long as necessary to support ongoing operations. Data may be deleted upon request or when no longer relevant.

7. Google API Services

This Application uses Google Chat API services. Use and transfer of information received from Google APIs adheres to the Google API Services User Data Policy, including the Limited Use requirements.

8. Security

We implement reasonable technical safeguards to protect data. Access is restricted to authorized personnel only.
